# 🧪 PRÁCTICA 3 — Auditoría UX  
## “El peor formulario del mundo”

### 🎯 Objetivo

Analizar un formulario **intencionalmente mal diseñado** para detectar problemas de **usabilidad, accesibilidad y diseño de interfaz**.

El objetivo de esta práctica es **desarrollar pensamiento crítico de diseño**, identificando errores comunes antes de aprender a corregirlos.

⚠️ **Importante:**  
En esta práctica **NO debes modificar el código del formulario.**

---

# 📋 Instrucciones

1. Abre el archivo **HTML del formulario** incluido en este repositorio.
2. Analiza el formulario desde la perspectiva de **experiencia de usuario (UX)**.
3. Identifica problemas de diseño, navegación o claridad.
